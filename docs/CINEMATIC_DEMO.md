# QuoteX motion judge film

The primary submission film presents working QuoteX evidence through a frame-accurate Remotion
composition. It combines kinetic typography, spring-timed camera moves, animated tool paths,
counting commercial data, real application clips, and a closing architecture pullback.

The sound layer uses scene-timed Qwen Cloud CosyVoice narration, the user-provided
`MA_Awesomemusic_ModernInterior.wav` track, and deterministic interface effects. Typewriter
clicks, switches, shutters, success tones, impacts, and scene whooshes are synchronized in source
code. Music levels ramp down smoothly beneath every spoken scene.

No frames, music, branding, or copy from the visual reference are reused.

## Reproduce

Prepare assets and inspect the timeline:

```bash
npm run motion:studio
```

Render a 12-second motion and audio check:

```bash
npm run render:motion-preview
```

Render nine judge-view frames:

```bash
npm run render:motion-stills
```

Render the final film:

```bash
npm run render:motion-demo
```

Inspect any difficult section without rerendering the whole film:

```bash
node demo/motion/render.mjs --segment 60 8
```

The renderer copies only the required local proof artifacts into an ignored public directory,
bundles the composition, renders H.264 through Remotion, applies final loudness normalization,
and writes a fast-start BT.709 standard-range delivery file. Process-scoped ports isolate
composition discovery from rendering, and the full export uses one deterministic browser worker.

  contained repeated 4 to 7 second low-motion spans at the same threshold.

- Integrated audio loudness: `-14.9 LUFS`.

- Audio delivery remains 48 kHz stereo after normalization.
- Left-minus-right analysis confirms the music and effects retain stereo information.
- The final file is tagged BT.709 limited range and decodes without errors.
- The quote overlay and live application evidence both show the recorded `$33,630` total.
- `npm test`, `npm run typecheck`, and `npm audit --omit=dev` pass.
